# Инструкция по добавлению изображений

## Структура папок

### `/images/background/`
Добавьте сюда изображения для фона сайта:
- `cnc-machining.jpg` - основное изображение токарной обработки (патрон, детали, станок)
- Рекомендуемый размер: 1920x1080 или больше
- Формат: JPG, PNG, WebP

### `/images/stands/`
Добавьте сюда изображения стоек Fanuc:
- `fanuc-2000c.jpg` - FANUC 2000C (1970-е)
- `fanuc-3000c.jpg` - FANUC 3000C (1970-е)
- `fanuc-system6.jpg` - FANUC System 6 (1976)
- `fanuc-system0.jpg` - FANUC System 0 (1980-е)
- `fanuc-0i-f.jpg` - FANUC 0i-F (2020-е)
- `fanuc-30i-b.jpg` - FANUC 30i-B (2020-е)

Рекомендуемый размер: минимум 800x600px
Формат: JPG, PNG, WebP

## Примечание
Если изображения не будут найдены, компоненты автоматически покажут иконки-заглушки.
